# 基于人脸识别的企业考勤管理系统

> 个人的毕业设计，提供了人脸识别考勤的简易管理系统。

## Build Setup

```bash

# 进入项目目录
cd vue-company-attendance-system

# 安装依赖
npm install

# 建议不要直接使用 cnpm 安装以来，会有各种诡异的 bug。可以通过如下操作解决 npm 下载速度慢的问题
npm install --registry=https://registry.npm.taobao.org

# 启动服务
npm run dev
```

## 发布

```bash

# 构建生产环境
npm run build:prod
```
